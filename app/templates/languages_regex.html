Languages and Regular Expressions

What is a Language in Automata?

A language is a set of strings (words) made from an alphabet (a finite set of symbols).

Example: If the alphabet Σ = {a, b}, then some strings are "a", "ab", "bba", etc.

A language can be finite (e.g., L = {ab, ba}) or infinite (e.g., all strings of any length over {a, b}).

Key Terms

  • Alphabet (Σ): Set of symbols, e.g., {a, b}.
  • String: Sequence of symbols, e.g., "abba".
  • Language (L): Set of strings over Σ, e.g., L = {ab, aabb, b}.
  • Empty String (ε): String of length 0.

Operations on Languages

  • Union (L₁ ∪ L₂): All strings in L₁ or L₂.
  • Concatenation (L₁L₂): All strings formed by joining a string from L₁ with one from L₂.
  • Kleene Star (L): All strings made by concatenating zero or more strings from L (includes ε).
  • Kleene Plus (L+): Like L*, but excludes ε (at least one string from L).

What is a Regular Expression?

A regular expression is a formula that describes a regular language using symbols and operators. It is a way to write patterns for matching strings.

Basic Regular Expression Syntax

  • ε: Represents the language containing only the empty string.
  • ∅: Represents the empty language (no strings).
  • a: Represents the language {a}.
  • r + s: Union (either r or s).
  • rs: Concatenation (r followed by s).
  • r*: Kleene star (zero or more repetitions of r).
  • r+: One or more repetitions of r.

Examples

Regular Expression Language Description Example Strings

Regular Expressions and Languages: The Connection

  • Every regular expression defines a regular language.
  • Every regular language can be described by a regular expression.
  • Regular expressions and finite automata are equivalent in power.

    ε, a, aa, aaa.
  • 2. If L = {ab}, what is L up to length 4?
    ε, ab, abab, ababab, abababab.
  • 3. Regular Expression for all binary strings (Σ = {0, 1}):
    (0+1)*.
  • 4. Is "ababab" in {ab}*?
    Yes, because it is "ab" repeated three times.
